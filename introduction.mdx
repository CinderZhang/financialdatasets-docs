### Overview
We have data for <b>15,000+</b> public companies, going back <b>30+</b> years.

You can use our data to power your AI financial agents, portfolio management tools,
stock analysis platforms, quantitative trading algorithms, and more.

You are welcome to save any data that you receive from our API.  Our platform is built by developers for developers.

### Getting Started
You can create an account and get an <b>API key</b> at [financialdatasets.ai](https://financialdatasets.ai).

For testing purposes, data for the following tickers is free: AAPL, GOOGL, MSFT, NVDA, TSLA.

We have outlined a few code examples below to get you started.

### Code Examples

<CodeGroup>

```python GET Income Statements
import requests

ticker = 'NVDA'     # stock ticker
period = 'annual'   # possible values are 'annual', 'quarterly', or 'ttm'
limit = 10          # number of statements to return

# create the URL
url = f'https://api.financialdatasets.ai/financials/income-statements?ticker={ticker}&period={period}&limit={limit}'

# add your API key to the headers
headers = {
    "X-API-KEY": "your_api_key_here"
}

# make API request
response = requests.get(url, headers=headers)

# parse income_statements
income_statements = response.json().get('income_statements')
```
